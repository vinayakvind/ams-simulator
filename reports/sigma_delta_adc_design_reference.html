Architecture

Power Tree and Control Partition

Analog input is integrated against a one-bit feedback DAC, quantized into a bitstream, and then low-pass filtered or decimated to reconstruct the in-band signal. The indexed wrapper keeps that architecture reusable across future implementations and technologies.

DomainVoltageSourceDrives
VDD3.3Vexternal supplyintegrator, quantizer, DAC model, decimation path
VINanalog stimulus centered in converter rangeanalog input sourcemodulator summing node
VREF0V to 2.5Vinternal or external reference pairone-bit feedback DAC swing

Bring-Up Order

  • Establish the reference endpoints before enabling the modulator loop.
  • Start the sampling clock so the integrator and quantizer enter a stable noise-shaping regime.
  • Observe the bitstream and filtered output over a long enough time window to assess behavior.

Control Path

  • Summing pathAnalog input and feedback DAC output form the modulator error signal.
  • Noise-shaping pathThe integrator accumulates low-frequency error while the one-bit quantizer emits the high-rate bitstream.
  • Reconstruction pathThe filtered or decimated output estimates the in-band analog value.

Architecture

The imported top-level reference models a first-order integrator, one-bit quantizer, feedback DAC, and simple output filtering in one reusable netlist.

  • Form the modulator errorSubtract feedback DAC output from the analog input at the summing node.
  • Integrate and quantizeThe integrator accumulates low-frequency error and the quantizer emits the one-bit bitstream.
  • Reconstruct the signalFilter or decimate the bitstream to recover the in-band output estimate.
